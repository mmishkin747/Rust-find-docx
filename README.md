# rust-find-docx
This is CLI program for find and open file docx

## Usage

for exaples:

```sh
rfd tests/input/hello.docx 
```
```sh
rfd test/input/hello.docx -p Test
```
```sh
rfd test/input/hello.docx -p test -i
```
```sh
rfd test/input/
```

## Running the tests

There are six tests.

```sh
cargo test
```
